# Group Fitness

PulsePal is a lightweight social fitness task app for creating wellness goals, tracking daily completion, and sharing completed goals to a simple social feed.

## Features

- Dashboard with active and completed goals
- Instant goal suggestions while creating a new goal
- Local browser persistence with `localStorage`
- Completion tracking with timestamped history
- Animated completion check/fade transition
- Social feed with mock posts, sharing, and reactions
- Responsive layout for mobile and desktop

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.
